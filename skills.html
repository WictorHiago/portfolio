HardSkills

MongoDB é um banco de dados NoSQL onde cada registro é um documento composto de pares chave-valor que são semelhantes aos objetos JSON (JavaScript Object Notation). O MongoDB é flexível e permite que seus usuários criem esquemas, bancos de dados, tabelas, etc. Documentos identificáveis ​​por uma chave primária compõem a unidade básica do MongoDB. Depois que o MongoDB estiver instalado, os usuários também poderão usar o shell do Mongo. O shell Mongo fornece uma interface JavaScript através da qual os usuários podem interagir e realizar operações CRUD.


Express é apenas uma estrutura de back-end para Node.js. Node.js é apenas um ambiente de tempo de execução que pode executar java-script. Express é uma estrutura JavaScript que fornece um conjunto robusto de recursos para aplicativos da Web e móveis. O Express fornece uma camada fina de recursos fundamentais de aplicativos da Web, sem obscurecer os recursos do Node.js.


O Node.js fornece um ambiente JavaScript que permite ao usuário executar seu código no servidor. Gerenciador de pacotes de nós, ou seja, o npm permite que o usuário escolha entre milhares de pacotes gratuitos (módulos de nó) para usar no aplicativo Ele usa o mecanismo V8 do chrome, que é um tempo de execução JavaScript de código aberto escrito em C++, que pega o código JS e o compila em código de máquina.


O MySQL é um sistema gerenciador de banco de dados relacional de código aberto usado na maioria das aplicações gratuitas para gerir suas bases de dados. O MySQL utiliza a linguagem SQL (Structure Query Language – Linguagem de Consulta Estruturada), que é a linguagem mais popular para inserir, acessar e gerenciar o conteúdo armazenado num banco de dados.
